# 考试系统
## 技术说明
采用技术包括：springboot、redis、shiro、thymeleaf、mybatis、mysql。
以idea为开发工具。
以maven构建项目。
## 项目说明
为简易考试系统，仅包含学生端。可进行考试。考题需在数据库自行设置。数据库建的很完善，后续可以添加很多功能，也可以去美化界面。
